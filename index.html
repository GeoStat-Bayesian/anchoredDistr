Welcome to anchoredDistr

This R package handles the post-processing from the Bayesian Uncertainty Characterization Framework MAD#: The Method of Anchored Distributions. To learn more about MAD#, visit our Codeplex site.

Installation

For easy installation, install and load the devtools package from CRAN, then call install_github("hsavoy/anchoredDistr").

Examples

See the anchoredDistr vignette for an introduction to the R package.
